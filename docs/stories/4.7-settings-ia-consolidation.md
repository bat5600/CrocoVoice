## Story
**As a** user,  
**I want** Profile and Billing to be consolidated with Settings inside the app with clear sub-navigation,  
**so that** I can find account and transcription settings easily.

## Acceptance Criteria
1. There is a single entry point in the hub/dashboard (in-app) for Settings/Account management (no separate Electron window).
2. The Settings surface includes sub-navigation (e.g., horizontal tabs) for: Profile, Billing, Transcription, and Advanced (exact naming can be adjusted).
3. Existing settings remain accessible (no lost controls); underlying setting keys remain stable.
4. Billing actions (upgrade/manage subscription) remain available and keep existing loading/error UX patterns.
5. Navigation is keyboard-accessible and does not introduce layout flashes/jumps.
