## Story
**As a** user,
**I want** to export transcriptions in common formats,
**so that** I can share and reuse results outside CrocoVoice.

## Acceptance Criteria
1. I can export a transcript as `txt`, `md`, or `json`.
2. Exports optionally include timestamps (per segment or per paragraph).
3. Export works for both live dictations and file uploads.
4. Exports run locally and do not upload data without consent.
5. Export uses a save dialog with a sensible default filename (`{date}-{title}.{ext}`).
6. Exports exclude sensitive context fields by default (ax/textbox/screenshot) unless explicitly enabled.
7. Export failures show clear error messages and do not crash the app.
