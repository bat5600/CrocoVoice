## Story
**As a** user,
**I want** to export transcriptions in common formats,
**so that** I can share and reuse results outside CrocoVoice.

## Acceptance Criteria
1. I can export a transcript as `txt`, `md`, or `json`.
2. Exports optionally include timestamps (per segment or per paragraph).
3. Export works for both live dictations and file uploads.
4. Exports run locally and do not upload data without consent.
5. Export failures show clear error messages and do not crash the app.

## Tasks / Subtasks
- [ ] Add export actions in history detail view (AC: 1,3)
- [ ] Implement export formatters (txt/md/json) (AC: 1)
- [ ] Add optional timestamp inclusion (AC: 2)
- [ ] Ensure exports are local-only and privacy-safe (AC: 4)
- [ ] Add error handling and user feedback (AC: 5)

## Dev Notes
- JSON export should include metadata (app, date, language, duration, source).
- Timestamp granularity can be coarse (e.g., per paragraph) for v1.

### Testing
- Manual: export from a live dictation and from an uploaded file.
- Manual: verify timestamp toggle on/off.
